# CLOS | Founder Series Agent Registry (v0.1-alpha)

This document defines the 37 specialized agentic nodes comprising the **Cognitive Life Operating System (CLOS)**. Each agent is a discrete inference loop optimized for a specific cognitive primitive.

## Swarm Topology
- **Executive Node:** The "Mirror" (User-Centric)
- **Communication:** Model Context Protocol (MCP) + gRPC
- **Inference:** Local-first (Apple Silicon / Metal)
- **Context:** Differential Privacy-hardened RAG

## Interaction Protocols
Agents communicate via **Semantic Handshakes**. No agent has full visibility of the user's secure enclave; they only access specific "slices" of context required for their specific function. This "Principle of Least Privilege" is the core of CLOS privacy.
