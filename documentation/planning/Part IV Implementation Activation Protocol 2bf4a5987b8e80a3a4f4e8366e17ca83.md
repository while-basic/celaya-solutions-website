## EMERGENCY PROTOCOLS

### **When You Miss a Day**

**DON'T:**

- Beat yourself up
- Abandon the system
- Try to "make up" missed work
- Spiral into "I've failed" thinking

**DO:**

- Acknowledge what happened (CLOS journal)
- Identify cause (too ambitious? External event?)
- Resume immediately next day
- Adjust if pattern emerges

**RULE:** One missed day = Resume. Two consecutive = Review and adjust. Three consecutive = Major system problem, redesign needed.

### **When You're Sick**

**MINIMUM SYSTEM:**

- Morning journal (if possible)
- Rest as primary job
- Evening journal (if possible)
- No guilt about not working

**CLOS tracks illness patterns for prediction/prevention.**

### **When Life Explodes**

**CRITICAL-ONLY MODE:**

```
Maintain ONLY:
1. Sleep schedule
2. One journal per day
3. One social connection per week

Everything else: PAUSE

Resume full system when crisis passes.

```
